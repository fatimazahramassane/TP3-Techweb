Google Homepage (HTML & CSS) 
## Description
This project is a simple reproduction of the Google homepage created using HTML5 and CSS3. The main objective is to practice web page structure, HTML elements organization, and CSS styling. This project is ideal for beginners in web development.
Technologies Used
- HTML5 (Page structure)
- CSS3 (Styling and layout)
- Local images (logo, icons, menu, etc.)
